# 💜 YAS.DEV | Portfolio Toolbox

Bem-vindo ao meu espaço pessoal de desenvolvimento e design. Este projeto é um portfólio dinâmico que une a robustez técnica da **Análise e Desenvolvimento de Sistemas** com a estética vibrante do **Visual Branding**.

---

## 🚀 Sobre o Projeto
Este site foi construído para ser uma central das minhas habilidades e projetos. Como profissional de ADS residente em São Paulo, foquei em criar uma interface de alta performance com uma identidade visual forte (e sim, totalmente roxa).

### 🛠️ Tecnologias Utilizadas
- **Front-end**: HTML5 semântico e CSS3 (Flexbox/Grid).
- **Lógica**: JavaScript para manipulação de DOM e efeitos de terminal.
- **Design**: Identidade visual desenvolvida com foco em UI/UX e Branding Visual (Canva/CapCut).
- **DevOps**: Controle de versão via Git e hospedagem no GitHub Pages.

---

## 📁 Estrutura de Navegação
O site está dividido em quatro módulos principais:
1. **Home**: Impacto visual e proposta de valor.
2. **Tools**: Minha stack tecnológica, incluindo Java, JavaScript e ferramentas de design.
3. **Projects**: Vitrine de trabalhos reais, como o projeto de branding **Divino Grão**.
4. **Terminal**: Uma interface interativa que simula um console, destacando meu background acadêmico.

---

## 🎨 Identidade Visual
A estética "Dark Mode" com gradientes roxos reflete uma obsessão de seis anos por essa cor, aplicada de forma estratégica para garantir legibilidade e modernidade.
